AntWave

Fast, professional RF, microwave, and antenna engineering toolkit.
All calculations run locally — no network required.

Core Features

  • Link Budget Analysis — Microwave, satellite, and radar link calculations with complete system parameters.
  • Antenna Metrics — Gain, beamwidth, G/T, noise temperature, pointing & surface accuracy loss.
  • Atmospheric Modeling — Interactive charts with real-time HUD readout. Sky noise and attenuation data based on ITU-R P.372/P.676/P.838 (1 GHz – 1 THz).
  • Transmission Line Tools — Characteristic impedance, waveguide parameters, skin depth.
